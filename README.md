#Tic-Tac-Toe Game
>Tic-Tac-Toe is a simple two-player strategy game played on a 3×3 grid.
>Players take turns marking:
X → Player 1
O → Player 2
>The objective is to form a line of three identical symbols in:
1.Row
2.Column
3.Diagonal
2. Technologies Used
a) Programming Languages
>Different languages can be used depending on the platform:
1.Python → Console-based or GUI-based applications
2.JavaScript → Web-based interactive games
Java / C++ → Desktop and mobile applications
b) Frontend Technologies (User Interface)
Used in web-based versions:
1.HTML
>Creates the 3×3 grid structure
Defines buttons or cells
2.CSS
>Designs layout (grid alignment)
Adds colors, animations, and styling
3.JavaScript
>Handles user clicks
Updates board dynamically
Checks winner logic
c) GUI Libraries (Desktop Applications)
>Used to create graphical interfaces:
1.Tkinter (Python) → Simple GUI apps
2.PyQt / Kivy → Advanced UI
3.Java Swing / JavaFX → Java-based UI
d) Artificial Intelligence (Optional)
Used for Player vs Computer mode:
1.Minimax Algorithm
Evaluates all possible moves
Chooses optimal move
Guarantees best outcome
2.Alpha-Beta Pruning
Reduces unnecessary calculations
Speeds up decision-making
e)Data Structures
1.Array/List
Stores board positions (size = 9)
2.Variables
Track current player
Track game status
